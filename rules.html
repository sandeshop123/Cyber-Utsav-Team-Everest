Quiz Rules

1. Time Limit

You will have a set time to complete each question. Answer quickly to get the best score!

2. Multiple Choice

Each question will have multiple answers, but only one is correct. Choose wisely!

3. Scoring

Points will be awarded for each correct answer. No points will be deducted for incorrect answers.

4. Quiz Topics

You can select from a variety of topics. Choose the one you feel most confident in.

5. No Skipping

Each question must be answered before moving on to the next one.
